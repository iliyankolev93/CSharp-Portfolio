# Expense Tracker Console App (C#)

A console-based personal finance tracker built with C# and .NET.

This project demonstrates working with collections, JSON file handling and basic data analysis.

## Features
- Add new expenses
- View all expenses
- Filter by category
- Calculate total spending
- Delete expenses
- Save and load data (JSON)

## Technologies
- C#
- .NET
- Console Application
- LINQ
- JSON Serialization
- File I/O
- Object-Oriented Programming (OOP)
- Git & GitHub

## Project Structure
- Program.cs - Application entry point
- Models/Expense.cs - Expense model
- Services/ExpenseManager.cs - Business logic
- Data/ExpenseRepository.cs - Data storage

## How to Run
1. Clone the repository:
```bash
git clone https://github.com/iliyankolev93/CSharp-Portfolio.git
```
2. Open the project in Visual Studio
3. Run with **Ctrl + F5**
