# ExpenseWebApp (ASP.NET Core MVC)

ExpenseWebApp is a web application for managing personal expenses using ASP.NET Core MVC and SQL Server.

This project demonstrates building a CRUD web application with Entity Framework Core and SQL Server integration.

## Features
- Create, edit and delete expenses
- Store data in SQL Server database
- Entity Framework Core
- Server
- Responsive UI with Bootstrap

## Technologies
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Razor Views
- Bootstrap
- jQuery
- C#

## Project Structure
- Controllers - Application logic
- Models - Data models
- Data - Database context
- Views - User interface
- Migrations - Database schema
- wwwroot - Static files

## How to Run
1. Clone the repository:
```bash
git clone https://github.com/iliyankolev93/CSharp-Portfolio.git
```
2. Open the solution in Visual Studio
3. Configure database connection in `appsettings.json`
4. Apply migrations:
```powershell
Update-Database
```
5. Run the project
